# PZA PARTY :: UNDERGROUND SLICE SOCIETY

Mystery popup pizza cult. EST. 2021.

## About

PZA PARTY is an underground popup pizza experience featuring handcrafted 10" pies with creative flavor combinations. Each drop is a secret event with locations revealed 24 hours before.

## Website

Single-page site with a retro cyberpunk/zine aesthetic featuring:
- Next drop information and event details
- Current menu with 5 signature pizzas
- Past drops archive
- Instagram integration (@kitchen_of_conner)

## Tech Stack

- Pure HTML/CSS (no build process required)
- Responsive design
- Custom animations and glitch effects
- VT323 and Courier Prime fonts
